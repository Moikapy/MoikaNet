# MoikaNet

> A  Webbrowser Using VueJS + Electron + Blockchain

>Status: Development

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm start

# build electron application for production
npm run build


```
